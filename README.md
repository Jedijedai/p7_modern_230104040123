Proyek ini merupakan hasil pengerjaan tugas Praktikum 7 Mobile Programming "Menerapkan Desain UI Modern" oleh Husni Majedi (NIM: 230104040123). Aplikasi dibangun menggunakan Android Studio dengan basis Kotlin dan Jetpack Compose, mengimplementasikan Material Design 3 untuk menciptakan antarmuka yang konsisten dan menarik. Fitur utama aplikasi mencakup halaman Login dengan validasi pengguna, halaman Home yang dilengkapi fungsi pencarian mata kuliah secara dinamis, halaman Profil dengan tata letak informasi akademik yang modern, serta halaman Pengaturan yang memfasilitasi perpindahan antara Light Mode dan Dark Mode. Navigasi antar halaman dikelola menggunakan Bottom Navigation Bar yang intuitif, memastikan pengalaman pengguna yang mulus saat berpindah fitur.
